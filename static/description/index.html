🤖 Perfect HR — AI Chatbot

AI-Powered Sales & Support Assistant

Transform your website into a 24/7 intelligent sales engine with the Perfect HR AI Chatbot — powered by local Ollama AI (Mistral 7B, LLaMA 3, Phi-3).

✨ Key Features

💬 Intelligent Chat Widget

  • Premium floating widget with glassmorphism design
  • Mobile responsive (full-screen on mobile)
  • Typing indicator & smooth animations
  • Session persistence across page navigations

🧠 Local AI Engine (Ollama)

  • Mistral 7B, LLaMA 3, Phi-3 support
  • RAG with FAISS vector search
  • No cloud APIs needed — runs 100% locally
  • Pull models directly from Odoo Settings

📚 Knowledge Base

  • Upload product info, FAQs, pricing, case studies
  • Auto-embedding pipeline with versioning
  • Kanban & list views for easy management

🎯 Lead Capture & CRM

  • Inline lead capture form in chat
  • Auto-create CRM leads with chat transcript
  • AI-based lead qualification scoring
  • Sales team assignment

📊 Analytics Dashboard

  • Conversation & lead KPIs
  • Conversion rate tracking
  • Response time analytics
  • Intent analysis & trends

🔒 Security & Compliance

  • Role-based access (User / Admin)
  • Data runs locally — no cloud leakage
  • CSRF protection on all endpoints
  • Session-based access control

⚙️ Requirements

  • Odoo CE v18
  • Ollama installed and running (https://ollama.ai)
  • At least one model pulled: ollama pull mistral
  • Optional: pip install faiss-cpu sentence-transformers for RAG
